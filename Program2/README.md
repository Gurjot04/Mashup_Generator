This is a web-based mashup generator built using Flask.  
The user provides:

- Singer Name
- Number of Videos
- Duration (in seconds)
- Email Address

The application:
1. Downloads songs from YouTube
2. Creates a mashup
3. Compresses the output into a ZIP file
4. Sends the ZIP file to the user via email
